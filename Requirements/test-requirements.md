# 要件定義書：ハムスター健康管理システム

## 1. プロジェクト概要
ハムスターの日常的な健康状態（主に体重）を記録・管理し、飼い主が異変にいち早く気づけるようにする。

## 2. 背景と課題
- **背景**: ハムスターは体が小さく、病気や体調不良を隠す習性がある。
- **課題**: 体重の微量な変化が健康のバロメーターとなるが、日々の変化を正確に把握し、比較することが難しい。

## 3. ターゲットユーザー
- ハムスターを飼育している個人。

## 4. 機能要件
- **体重記録機能**: 毎日の体重（グラム単位）を日付とともに記録できる。
- **推移グラフ表示**: 記録された体重データを時系列でグラフ化し、視覚的に変化を確認できる。
- **アラート通知**: 設定した閾値以上の急激な体重減少・増加があった場合に警告を表示する。
- **複数個体管理**: 複数のハムスターを飼育している場合、個体ごとにプロファイルを分けて管理できる。
- **メモ機能**: 給餌内容や活動状況などのメモを残せる。

## 5. 非機能要件
- **操作性**: 毎日の計測時に素早く入力できるシンプルなUI。
- **データ永続性**: 記録したデータが適切に保存され、後から振り返ることができる。

## 6. 将来的な拡張性
- 餌の管理機能
- 通院記録
- 飼育コミュニティとの連携
