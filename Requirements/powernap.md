# PowerNap

## Overview
昼寝のつもりが深く寝入りすぎてしまい、起きた後に体がだるくなったり、夜の睡眠に支障が出たりすることを防ぐためのシンプルで使いやすいタイマー。

## Target User
昼間の休憩時間に効率的に休息を取りたい社会人や学生。

## Problem Statement
昼寝は短時間（15分〜20分程度）であればリフレッシュ効果が高いが、30分を超えて深い眠りに入ってしまうと、起床時の不快感（睡眠慣性）が強くなり、逆効果になることがある。

## MVP
昼寝に最適なプリセット時間を選択し、アラームを鳴らす最小限の機能。

### Feature List
- プリセットタイマー選択（15分, 20分, 30分, 45分, 60分）
- カウントダウン表示
- アラーム（鳴動・停止）

## Screens

### Home
- タイマーのプリセット選択ボタン一覧
- 動作中のカウントダウン表示と停止ボタン

### Settings
- アラーム音の選択
- 音量設定

## Data Model

### TimerSettings
- id: string
- presetDuration: number (minutes)
- alarmSound: string

## API

### GET /api/settings
タイマー設定（アラーム音等）の取得

### POST /api/settings
タイマー設定の更新

## Non Functional Requirements

### Security
- ローカルデバイス上での動作。特別なユーザー情報の収集は不要。

### Performance
- バックグラウンドでも正確にタイマーが動作すること。
- 操作に対する即時のレスポンス。

## Future Features
- 昼寝の履歴記録
- 統計表示（累積昼寝時間など）
- おすすめの昼寝時間の提案（最新の研究に基づく）
- 睡眠の深さを検知して最適なタイミングで起こすスマートアラーム機能

## Monetization
- 広告表示（無料版）
- 有料版での広告非表示・特殊なアラーム音の追加
